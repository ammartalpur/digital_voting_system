About Our Project

Digital Voting System

This Voting System Project is a C++ console application that simulates an election process, allowing admins to add candidates and users to cast votes. It displays real-time results, handles input validation, and provides a clean interface. The project demonstrates key programming concepts like dynamic memory allocation and control structures, offering a practical learning experience.
